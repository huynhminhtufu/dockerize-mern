## Dockerize development environment for MERN stack

### How to intall

- Install `DockerCE`
- Run command: `docker-compose up`

### Server: Node.js

- Express server
- Nodemon watcher
- Morgan logger
- Sequelize ORM with MySQL driver

### Webapp: React

- `create-react-app` v2
- Webpack
- Hot module reload

### Database: MySQL 5.7

- Built in database server
